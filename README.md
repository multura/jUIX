# JUIX
Продолжение jUI.

## Функции
* **Браузер** - стандартный браузер DevelNext
* **Центр обновлений** - проверяйте вашу сборку jUIX на наличие обновлений
* **Блокнот** - блокнот, без возможности сохранения
* **marketplace** - онлайн магазин PWA-приложений
